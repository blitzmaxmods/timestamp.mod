# Ctime.day()

## Syntax
```
Ctime.day:Int()
``` 

## Description

Ctime.Int() returns the day of the month 

## Return Value
Ctime.day() returns an Int

## Dependencies
* [BlitzMaxNG](https://blitzmax.org)

## Threadsafe
Has not been evaluated, but all the C functions used have been confirmed
