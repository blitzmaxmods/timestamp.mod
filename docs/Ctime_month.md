# Ctime.month()

## Syntax
```
Ctime.month:Int()
``` 

## Description

Ctime.Int() returns the month number of the date. 

## Return Value
Ctime.month() returns an Int

## Dependencies
* [BlitzMaxNG](https://blitzmax.org)

## Threadsafe
Has not been evaluated, but all the C functions used have been confirmed
